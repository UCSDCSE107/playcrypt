Welcome to PlayCrypt's documentation!
============================================

A Python library for introductory theoretical cryptography courses. The
following is all you need to get started using our framework. In the
introduction you will find a brief introduction to the project. Following that
there is a `Getting Started <quick_start.html>`_ guide which is a quick way to
get up and running.
